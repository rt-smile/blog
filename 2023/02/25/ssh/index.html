ssh

什么是ssh？

安全外壳协议（Secure Shell Protocol，简称SSH） 是一种加密的网络传输协议，可在不安全的网络中为网络服务提供安全的传输环境。
SSH通过在网络中建立安全隧道来实现SSH客户端与服务器之间的连接。
SSH以非对称加密 实现身份验证。
最流行的SSH实现：OpenSSH
常见用途：远程登录系统。

ssh有关的命令（工具）

  • ssh-keygen
    用于生成密钥
    如：ssh-keygen -t ed25519 -C “your_email@example.com“
  • ssh-agent
    是一个密钥管理器, OpenSSH authentication agent.
    如： eval “$(ssh-agent -s)”
  • ssh-add
    adds private key identities to the OpenSSH authentication agent
    如 ssh-add -l // 查看密钥是否被加载
    ssh-add ~/.ssh/id_ed25519 // 添加密钥
  • ssh-ssh-copy-id
    use locally available keys to authorise logins on a remote machine
    如 ssh-copy-id -i ~/.ssh/vpn.pub root@1.1.1.1 //把公钥添加到远程服务器
    • 免密登陆的两种方式

关于.ssh文件夹

  • known_hosts
    存储用户访问的主机的公钥
    每次通过SSH连接到新的远程服务器时，系统会提示是否储存该远程主机
    避免收到dns-hijack(域名劫持)之类的攻击，
  • config
    配置文件
    例如，每次重启都要重新 ssh-add 密钥 到 ssh-agent上。在config设置好后，就不用了。
    • SSH config
    • Multiple SSH Keys on same client
