BOM&&DOM

window

window.innerHeight

浏览器窗口的视口（viewport）高度。
如果有水平滚动条，也会包括进去！

window.innerWidth

浏览器窗口的视口（viewport）宽度。
如果有垂直滚动条，也会包含进去！
说明： 如果需要获取去除滚动条的宽度，访问跟元素<html>的clientWidth属性.

Window.outerHeight

浏览器窗口的高度

Window.outerWidth

浏览器窗口的宽度

window.screenLeft 和 window.screenX

前者是后者的别名
返回浏览器左边界到桌面左边界的水平距离。

window.screenTop 和 window.screenY

前者是后者的别名
返回浏览器顶部到桌面顶部的垂直距离

window.pageXOffset 和 window.scrollX

前者是后者的别名
返回页面水平滚动的距离

window.pageYOffset 和 window.scrollY

前者是后者的别名
返回页面垂直方向滚动的距离

element

element.clientLeft

The width of the left border of an element in pixels.
元素左边框的宽度

element.clientTop

The width of the top border of an element in pixels.
元素上边框的宽度

element.clientHeight

返回元素的高度 (即本身有定义height,则为height的高度；否则为元素里面的内容撑开的高度)
在根元素（<html>元素）或怪异模式下的<body>元素上使用 clientHeight 时，该属性将返回(viewport’s height)视口高度（不包含任何滚动条）。

element.clientWidth

返回元素的宽度 (即本身定义的width,否则是元素里面的内容撑开的宽度)
在根元素（<html> 元素）或怪异模式下的<body> 元素上使用 clientWidth 时，该属性将返回(viewport’s height)视口宽度（不包含任何滚动条）。

element.scrollHeight

返回元素里内容的高度
即元素里的内容高度小于或等于（内容撑开容器的height）元素的高时，scrollHeight等于元素的高。当内容的高大于元素时，则为内容的高
用例：判断内容是否滚动到底

element.scrollWidth

返回元素里内容的宽度
解释：与scrollHeight一样

element.scrollTop

获取或设置内容垂直滚动（卷起来）的距离，即可见内容顶部到元素整个内容顶部的距离
如果元素的内容没有产生垂直滚动条，则 scrollTop 为0

element.scrollLeft

获取或设置内容水平（向左）滚动的距离

HTMLElement

HTMLElement.offsetHeight

返回元素的高度（类似于element.clientHeight，只不过HTMLElement.offsetHeight包括边框、水平滚动条和根元素返回的是总高度，而不是视图高度）

HTMLElement.offsetWidth

返回元素的宽度（类似于element.clientWidth，只不过HTMLElement.offsetWidth包括边框、垂直滚动条和根元素返回的是总宽度，而不是视图宽度）

HTMLElement.offsetLeft

返回元素左上角距离HTMLElement.offsetParent左边界的距离

HTMLElement.offsetTop

返回元素顶部上面到HTMLElement.offsetParent顶部下面的距离
